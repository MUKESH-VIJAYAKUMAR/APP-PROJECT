# Smart Parking Space Finder

**Smart Parking Space Finder** - a modern desktop application built with Java Swing that helps users create accounts, log in, and manage parking slot bookings.
The app features a clean UI with gradient themes, real-time slot updates, and a token-based parking system for secure and organized management.
Overview


# Features

This application simulates a mini smart-parking environment.
Users can:

- Create an account
- Log in securely
- View all available parking slots
- Book a slot using a unique generated token
- Release the slot when leaving
- Track occupied and available slots in real time
